# Code Problem 344

##Problem
####[Reverse String](https://leetcode.com/problems/reverse-string/)
Write a function that takes a string as input and returns the string reversed.

Example:
Given s = "hello", return "olleh".

- Node
- Grunt

```
npm install -g grunt-cli
```

```
npm install
npm test
```
