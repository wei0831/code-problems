# Code Problem 13

##Problem
####[Roman to Integer](https://leetcode.com/problems/roman-to-integer/)
Given a roman numeral, convert it to an integer.

Input is guaranteed to be within the range from 1 to 3999.

- Node
- Grunt
```
npm install -g grunt-cli
```

## How to Run?

```
npm install
npm test
```
